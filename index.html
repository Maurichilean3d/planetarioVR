Planetario VR Inmersivo

Sistema listo para explorar

Meta Quest: abre en HTTPS o localhost, pulsa Enter VR y mira alrededor. El gatillo cambia el planeta enfocado.

PC: arrastra para mirar, WASD para volar, Q/E para bajar/subir, rueda para avanzar/retroceder.

Esta versión prioriza presencia: plataforma, polvo espacial cercano, nebulosas 360°, cometas, audio opcional y panel holográfico dentro de la escena.
